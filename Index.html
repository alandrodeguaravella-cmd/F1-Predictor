        How to add real images (optional)

        This project uses placeholders by default. If you want real team logos / driver faces / race flags:

        1. Create an assets/ folder next to index.html.
        2. Add files like assets/teams/ferrari.png, assets/drivers/verstappen.jpg, assets/flags/aus.png.
        3. Then update paths in app.js (search: ASSET_MODE).

        (Keeping it placeholder-only avoids copyright/usage headaches, but the hooks are ready.)
